# Screenshots

โปรดเพิ่มภาพหน้าจอของแอปพลิเคชันในโฟลเดอร์นี้

## รายการภาพที่ต้องการ:

1. **login.png** - หน้า Login
2. **dashboard.png** - หน้า Dashboard
3. **camera.png** - หน้า Camera Scanner
4. **vehicle_list.png** - หน้ารายการรถ
5. **statistics.png** - หน้าสถิติ
6. **settings.png** - หน้าตั้งค่า

## วิธีการถ่ายภาพหน้าจอ:

### Windows
- กด `Windows + Shift + S` แล้วเลือกพื้นที่

### Chrome DevTools
1. เปิด DevTools (F12)
2. Toggle device toolbar (Ctrl+Shift+M)
3. เลือก device (iPhone, Pixel, etc.)
4. คลิกขวา > Capture screenshot

### Flutter
```bash
# Run ในโหมด Debug
flutter run

# ถ่ายภาพหน้าจอผ่าน DevTools
# หรือใช้ปุ่ม camera icon ใน Flutter DevTools
```

## ขนาดภาพแนะนำ:
- **Width**: 250-400px (สำหรับแสดงใน README)
- **Format**: PNG
- **Resolution**: มาตรฐาน mobile (360x640 หรือ 375x667)
